# Changelog

## 0.1.0 - Initial MVP

- Python-only Smart Metadata Reader node
- Reads image metadata from PNG / JPEG / WEBP
- ComfyUI prompt/workflow parser
- A1111 / Forge parameters fallback
- Positive / negative extraction from selected sampler conditioning chain
- ControlNet / CR Apply ControlNet passthrough support
- StringFunction|pysssss / ShowText|pysssss / DeepTranslatorTextNode support
- Checkpoint / VAE / LoRA / sampler setting extraction
- Human-readable setting output
- Node options documented
- Regression tests included
